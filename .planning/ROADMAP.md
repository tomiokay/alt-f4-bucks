## Phase 1: Betting System

**Goal:** Add peer-to-peer FRC match betting with Polymarket-style UI, TBA/Statbotics integration, and auto-resolution

**Requirements:** MATCH-01, MATCH-02, MATCH-03, MATCH-04, BET-01, BET-02, BET-03, BET-04, BET-05, BET-06, BET-07, RES-01, RES-02, RES-03, RES-04, UI-01, UI-02, UI-03, UI-04, UI-05

**UI hint:** yes

**Success Criteria:**
1. User can browse upcoming FRC matches from TBA with team/event info displayed
2. User can create a bet (pick match, type, wager, opponent) and send it
3. Opponent can accept/decline; acceptance locks both wagers via escrow
4. Bets auto-resolve when TBA posts match results, winner gets payout
5. Polymarket-style UI with clean match cards and intuitive bet placement
6. Statbotics win probability shown on match cards
7. My Bets page shows full bet history with statuses

**Status:** Not started
